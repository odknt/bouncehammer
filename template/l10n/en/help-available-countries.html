available-countries

Kanadzuchi::Mail::Group reads etc/available-countries as YAML for getting the parsed results which the host group & the provider are correctly classified.

If the file does not exist, Kanadzuchi::Mail::Group load all of modules in the directory Kanadzuchi/Mail/Group/XX

To use

If you use the file, it should be saved as 'available-countries', Run /bin/cp ./available-countries-example ./available-countries, And Edit the file at the command line.

Value
[% tt_word.use %]
Load XX/YY.pm module in the directory Kanadzuchi/Mail/Group. Then it correctly classify host group & provider.
[% tt_word.disuse %]
Do not load the host group(by area) library, Then Host Group is "PC", Provider is "various" in parsed results.

XX = ccTLD
YY = Host Group: WebMail, Smartphone, & Cellphone.

MTA

Load only the module of its name in Kanadzuch/MTA/XX/ZZ.pm

XX = ccTLD
YY = Host Group: WebMail, Smartphone, & Cellphone.
ZZ = MTA Name
