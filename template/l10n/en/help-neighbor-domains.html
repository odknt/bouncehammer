neighbor-domains

Kanadzuchi/Mail/Group/ Neighbor.pm reads etc/neighbor-domains. If a provider name and a domain name are defined in the file, the hostgroup is classified as "neighbor", and the provider name is classified by the name of its provider.

To use

If you use the file, it should be saved as "neighbor-domains", Run /bin/cp ./neighbor-domains-example ./neighbor-domains, And Edit the file at the command line.
